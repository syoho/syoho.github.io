学习汇总｜Java学习路线

学习汇总｜Java学习路线


给自己制定的初步学习路线
01 / 02是现阶段的首要任务

01 成果制作

  • 目标：

    • 搭建个人博客网站
    • 完成两个简单的Java项目
    • 使用外语对Java项目进行复述讲解

  • 原因：

    • 记录：编程注定是个漫长且无边际的过程，将其中的细节作为记录，放置在博客上，也算是刻下人生的痕迹
    • 简历：先完成可视化的成果，并能用外语阐述；这样即便后续学习时间紧张，至少也能在简历上添上几笔
    • 证明：虽然项目有些稚嫩，但至少可作为自己的学习态度与行动力的证明

  • 内容：

    • 个人博客：Hexo框架 / Fluid主题
    • 项目01：タスク管理アプリケーション （全流程3遍：项目制作 / 代码原理 / 外语解释）
    • 项目02：SpringBoot+Vue全栈开发 （全流程3遍：项目制作 / 代码原理 / 外语解释）

02 基础巩固

  • Java基础：

    • 视频：「韩顺平零基础30天学会Java」
    • 书籍：「プロになるJava―仕事で必要なプログラミングの知識がゼロから身につく最高の指南書」

  • 外语练习：

    • Youtube相关视频（后续确定）

03 企业开发

  • Java框架：

    • Java Web
    • Spring 5
    • Spring MVC
    • MyBatis
    • Spring Boot 2
    • Maven

  • MySQL

  • Git

  • Linux

04 技术进阶

  • Redis（缓存）
  • RabbitMQ（消息队列）
  • Nginx（Web服务器）
  • Dubbo（微服务）
  • Docker（容器）

05 深入研究

  • 并发编程
  • JVM
